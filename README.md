# llma-cpu

## How To run ?

### step 1:

clone the repository 

``` bash
git clone https://github.com/srktheman000/llma-cpu.git
```


### step 2:

create a virtual enviornment

``` bash
conda create -n llma-cpu python=3.8 -y 
```

``` bash
conda  activate llma-cpu
```

``` bash
pip install -r requirements.txt
```
